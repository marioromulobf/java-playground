### Java - Mockito 5 POC

**Core Java - Mockito 5 - All features**

### Tech Stack
* Java 25 (LTS)
* Maven 3.x
* JUnit 6
* Mockito 5.22

### How to Build
```bash
mvn clean compile
```

### How to Run
```bash
mvn exec:java
```

Or run directly:
```bash
java -cp target/classes com.mariofernandes.javapoc.mockito.Main
```
