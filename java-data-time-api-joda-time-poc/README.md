### Java - Data Time API + Joda time POC

**Core Java - Data Time API + Joda time**

### Tech Stack
* Java 25 (LTS)
* Maven 3.x
* JUnit 6.1

### How to Build
```bash
mvn clean compile
```

### How to Run
```bash
mvn exec:java
```

Or run directly:
```bash
java -cp target/classes com.mariofernandes.javapoc.dtajt.Main
```
