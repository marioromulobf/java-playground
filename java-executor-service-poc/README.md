### Java Executor Service POC

**Core Java - Executor Service**

Demonstrates Java Executor Service capabilities:


### Tech Stack
* Java 25 (LTS)
* Maven 3.x
* JUnit 6

### How to Build
```bash
mvn clean compile
```

### How to Run
```bash
mvn exec:java
```

Or run directly:
```bash
java -cp target/classes com.mariofernandes.javapoc.execs.Main
```
