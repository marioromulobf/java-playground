### Java - JUnit 5 POC

**Core Java - JUnit 5 - All features**

### Tech Stack
* Java 25 (LTS)
* Maven 3.x
* JUnit 5.x

### How to Build
```bash
mvn clean compile
```

### How to Run
```bash
mvn exec:java
```

Or run directly:
```bash
java -cp target/classes com.mariofernandes.javapoc.junit.Main
```
